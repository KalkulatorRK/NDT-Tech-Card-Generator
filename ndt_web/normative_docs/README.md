# Нормативные документы

В этой папке хранятся тексты нормативных и методических документов,
используемых для расчётов в приложении «НК-Карта».

## Приоритетность загрузки

| Приоритет | Документ | Для чего используется |
|-----------|----------|----------------------|
| 🔴 Высокий | ГОСТ Р 50.05.07-2018 | Генератор техкарт РГК: таблицы чувствительности, схемы, источники, плёнки |
| 🔴 Высокий | НП-105-18 | Оценка качества: критерии допустимости дефектов по категориям |
| 🟡 Средний | НП-104-18 | Категории сварных соединений I–IV |
| 🟡 Средний | ГОСТ Р 59023.2-2020 | Типы сварных соединений, марки сталей |
| 🟢 Дополнительно | ГОСТ 7512-82 | Схемы просвечивания, формулы расчёта f, N, L |

## Как добавить документ

1. Скопируйте текст из PDF в текстовый файл (`.txt`)  
   **или** загрузите `.docx` / `.txt` файл напрямую в эту папку через GitHub
2. Назовите файл по шаблону: `ГОСТ_Р_50.05.07-2018.txt`
3. После загрузки сообщите — данные будут перенесены в код

## Что изменится после добавления документов

### После загрузки ГОСТ Р 50.05.07-2018
- Проверка и уточнение таблиц чувствительности (Приложение А)
- Уточнение требований к плёнкам, экранам, эталонам
- Добавление требований к обработке плёнки (п. 6–8)
- Уточнение критериев геометрической нерезкости

### После загрузки НП-105-18
- Проверка и уточнение критериев оценки дефектов (Таблица 1)
- Добавление балльной оценки по всем типам дефектов
- Уточнение требований по скоплениям пор
- Добавление критериев для поверхностных дефектов

### После загрузки ГОСТ 7512-82
- Уточнение таблиц TABLE_DATA_5A, 5B, 5G (схемы просвечивания)
- Проверка формул расчёта f, N, L для каждой схемы
- Добавление схем которых пока нет (3е, 3и)

## Формат текстовых файлов

Текст может быть скопирован «как есть» из PDF.  
Таблицы лучше сохранять с выравниванием пробелами или в виде:

```
Таблица 1 — Минимальная чувствительность
S, мм      | Класс А | Класс В | Класс С
до 5       |   1,5   |   2,0   |   3,0
св. 5 до 10|   1,0   |   1,5   |   2,0
...
```
