# Шаблоны технологических карт

В эту папку помещаются шаблоны (бланки) технологических карт в формате DOCX.

## Содержимое

| Файл | Описание |
|------|----------|
| `TC_тк 67 (образец отступов…).docx` | **Основной шаблон** — отступы таблиц от колонтитулов, разрывы блоков 4.3 / 6.9 / 7–10 |
| `Пример технологической карты радиографического контроля.docx` | Резервный шаблон |

## Как добавить шаблон

1. Подготовьте бланк техкарты в формате `.docx`
2. Загрузите файл в эту папку через GitHub или git
3. Файлы с префиксом `TC_` подхватываются генератором автоматически

## Примечание

Генератор заполняет готовый бланк `.docx` данными, сохраняя вёрстку шаблона
(шрифты, отступы, пустые абзацы между блоками, разрывы страниц).
