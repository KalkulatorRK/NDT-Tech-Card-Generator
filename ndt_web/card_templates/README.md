# Шаблоны технологических карт

В эту папку помещаются шаблоны (бланки) технологических карт в формате DOCX.

## Содержимое

| Файл | Описание |
|------|----------|
| `radiographic_template.docx` | Шаблон ТК радиографического контроля (ГОСТ Р 50.05.07-2018) |

## Как добавить шаблон

1. Подготовьте бланк техкарты в формате `.docx`
2. Загрузите файл в эту папку через GitHub или git
3. Укажите имя файла в настройках генератора

## Примечание

Генератор может работать в двух режимах:
- **С шаблоном**: заполняет готовый бланк `.docx` данными
- **Без шаблона**: создаёт документ с нуля (текущий режим)
